- Darknet training data directory structure (see `dataset/darknet` for details).
    ```text {linenos=table}
    darknet
    ├── class.names
    ├── gen_config.data
    ├── gen_train.txt
    ├── gen_valid.txt
    └── images
        ├── train
        └── valid
    ```
- Convert
    ```bash {linenos=table}
    darknet_to_coco --data_path dataset/darknet/gen_config.data
    ```
